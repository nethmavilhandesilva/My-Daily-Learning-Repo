So what is “AWS Docker”?

It usually means running Docker containers on AWS cloud services.

AWS provides services to run Docker containers easily:

Amazon ECS
→ Run and manage Docker containers without managing servers
Amazon EKS
→ Run containers using Kubernetes (more advanced)
AWS Fargate
→ Run containers without managing servers at all
